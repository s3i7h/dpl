## Installation

```sh
brew tap shu-pf/tap
```

```sh
brew install dpl
```

## Usage

To use this package, you first need to set an API key and Target Language.

### Config(Required)

```sh
dpl setting
```

### 1. Normal mode

```sh
dpl <text>
```

### 2. Interactive mode

```sh
dpl
```

## For Developers

### Execute

```sh
pnpm start
```

### Build

```sh
pnpm build
```
